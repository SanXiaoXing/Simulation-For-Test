# 输入参数

| 名称 | 类型 | 范围 | 字节数 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| 包头 | Int | AA55 | 2 | 帧起始标志 |
| 目标 | Int | 0~255 | 1 | 参数配置 |
| 长度 | Int | 0~65535 | 2 | 后续有效数据长度 |
| ImageCenterLon | Double | -180~180 | 8 | 图形中心经度 |
| ImageCenterLat | Double | -90~90 | 8 | 图形中心纬度 |
| ImageTarget_num | Int | 0~255 | 1 | 目标个数 |
| ImageTarget_max | Int | 0~255 | 1 | 最大目标数 |
| ImageTarget_id | Int | 0~255 | 1 | 唯一标识 |
| Type | Int | 0~255 | 1 | 目标类型 |
| ImageTarget_distance_m | Float | 0~65535 | 4 | 距离 |
| ImageTarget_azimuth_deg | Float | -180~180 | 4 | 方位 |
| ImageTarget_elevation_deg | Float | -90~90 | 4 | 俯仰 |
| ImageTarget_velocity | Float | -3000~3000 | 4 | 径向速度 |
| Confidence | Float | 0~1 | 4 | 置信度 |
| ThreatLevel | Int | 0~10 | 1 | 威胁等级 |
| StealthFlag | Int | 0/1 | 1 | 隐身标志 |
| TrackCmd | Int | 0~10 | 1 | 跟踪指令 |
| AzimuthMiss_deg | Float | -180~180 | 4 | 方位脱靶量 |
| ElevationMiss_deg | Float | -90~90 | 4 | 俯仰脱靶量 |
| TargetAzimuth_deg | Float | -180~180 | 4 | 目标方位 |
| FOVCenterAzimuth | Float | -180~180 | 4 | 视场中心 |
