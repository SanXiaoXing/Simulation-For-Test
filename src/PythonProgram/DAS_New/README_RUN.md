# DAS 模拟器运行说明

## 环境要求
- Python 3.8+
- PyQt5
- matplotlib
- numpy

## 文件列表
- `das_main.py`: DAS模拟器主程序（PyQt5界面 + 3D态势显示）。
- `das_protocol.py`: 通信协议定义。
- `das_config.json`: 配置文件（网络、传感器参数、初始目标）。
- `das_test_server.py`: 测试服务端（模拟显控端发送指令）。
- `DAS.py`: 原始逻辑参考（未使用）。

## 运行步骤

1. **启动测试服务端** (模拟显控端发指令):
   ```bash
   python das_test_server.py
   ```
   *注意：服务端会监听 6000 端口，并向 6001 发送控制指令。*

2. **启动 DAS 模拟器**:
   ```bash
   python das_main.py
   ```
   *注意：模拟器启动后会显示主界面，包含左侧控制区/列表和右侧3D态势图。它会监听 6001 端口接收指令，并向 6000 端口发送传感器数据。*

## 功能说明
- **界面**: 
  - 左侧：系统模式设置、目标列表显示。
  - 右侧：3D 空间态势显示（飞机位置、目标位置）。
- **通信**: 
  - 接收：解析显控发来的系统控制指令（模式、状态等）。
  - 发送：打包发送传感器状态、探测到的目标列表、视场信息等。
- **仿真**: 
  - 内置简单的目标运动逻辑。
  - 自动计算目标相对于飞机的方位、俯仰和距离。
